# Images du carousel de la page d'accueil

Ajoute ici les images a afficher dans le carousel de la page d'accueil.

Formats acceptes : `png`, `jpg`, `jpeg`, `webp`, `avif`, `gif`, `svg`.

Les fichiers sont tries par nom. Pour controler l'ordre, prefixe les fichiers :

```text
01-annonce.png
02-tournoi.webp
03-evenement.jpg
```

Apres ajout ou suppression d'image, relance le front ou refais un build.
